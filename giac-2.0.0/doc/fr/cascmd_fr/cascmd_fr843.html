9.4.10  Pour effacer les sorties : ClrIO

ClrIO n’a pas d’argument, il efface toutes les sorties faites dans le niveau où il a été tapé.
On tape :

a:=12

puis :

print("a=",a)

On obtient :

"a=",12 s’écrit en bleu et la réponse est 1

On tape :

print("a=",a);ClrIO()

On obtient :

(1,1) comme réponse
