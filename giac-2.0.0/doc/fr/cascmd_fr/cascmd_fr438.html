6.33.5  Borne supérieure des racines réelles positives d’un polynôme : posubLMQ

posubLMQ(P) renvoie une borne supérieure pour les racines positives de P par l’algorithme Akritas-Strzebonski-Vigklas’ Local Max Quadratic (LMQ). Cette borne n’est pas optimale.
On tape :
posubLMQ(x^3-7*x+7)
On obtient :
4
On tape :
posubLMQ(x^5+2*x^4-6*x^3-7*x^2+7*x+7)
On obtient :
4
On tape :
posubLMQ(x^3-x^2-2*x+2)
On obtient :
3
