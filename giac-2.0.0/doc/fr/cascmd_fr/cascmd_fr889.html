10.2.4  Tracer un repère : repere_2d frame_2d

repere_2d() frame_2d() trace le repère de écran de géométrie 2-d.
Vous pouvez effacer les axes (resp les faire réapparaitre) en cochant (resp décochant) Montrer les axes avec le bouton cfg de l’écran de géométrie.
Ces commandes n’ont pas de paramètre.
On tape :

repere_2d()
