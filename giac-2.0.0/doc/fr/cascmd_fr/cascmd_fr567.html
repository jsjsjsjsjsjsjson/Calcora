6.46.8  Matrice de Vandermonde : vandermonde

vandermonde a comme argument un vecteur de composantes xj.
vandermonde renvoie la matrice de Vandermonde correspondante : elle a pour kième ligne est le vecteur de composantes xjk (k=0..n−1).
Attention !
Xcas numérote les lignes et les colonnes à partir de 0.
On tape :

vandermonde([a,2,3])

On obtient (si a n’est pas affecté) :

[[1,1,1],[a,2,3],[a*a,4,9]]
