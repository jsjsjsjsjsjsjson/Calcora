6.62.5  Pour avoir le membre de droite d’une équation : right droit rhs

right ou rhs a comme paramètre une équation ou un intervalle.
right ou rhs renvoie le membre de droite de l’équationou de la borne droite de l’intervalle.
On tape :

right(2x-1=3)

Ou on tape :

rhs(2x-1=3)

On obtient :

3

On tape :

right(1..3)

Ou on tape :

rhs(1..3)

On obtient :

3
