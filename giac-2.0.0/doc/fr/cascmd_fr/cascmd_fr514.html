6.42.9  Insérer un élément dans une liste ou une chaine de caractères : insert

insert insère dans une liste, ou dans une chaine de caractèresà l’indice donné en 2nd argument, l’élément donné en 3ième argument.
insert renvoie une erreur si l’indice donné p est trop grand i.e. si p>dim(L).
Attention ! l’indice du premier élément est 0.
On tape :

insert([3,4,2],2,5)

On obtient :

[3,4,5,2]

On tape :

insert("342",2,"5")

On obtient :

"3452"

On tape :

insert([3,4,2],4,5)

On obtient :

insert([3,4,2],4,5)Erreur: Dimension invalide
