6.53.9  Base d’un sous espace vectoriel : basis

basis a comme argument la liste des composantes des vecteurs qui engendrent un sous espace vectoriel de ℝn.
basis renvoie une liste constituée des vecteurs d’une base de ce sous espace vectoriel.
On tape :

basis([[1,2,3],[1,1,1],[2,3,4]])

On obtient :

[[1,0,-1], [0,1,2]]
