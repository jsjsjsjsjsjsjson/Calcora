6.5.11  Concaténer des objets en une chaîne : cat

cat a comme paramètre une séquence d’objets.
cat concaténe ces objets en une chaîne de caractères.
On tape :

cat("abcd",3,"d")

On obtient :

"abcd3d"

On tape :

c:=5
cat("abcd",c,"e")

On obtient :

"abcd5e"

On tape :

purge(c)
cat(15,c,3)

On obtient :

"15c3"
